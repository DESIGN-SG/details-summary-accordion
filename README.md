# details-summary-accordion
HTML/CSS/JavaScriptでアコーディオンを作成しました。

### 目的
- `<details>` `<summary>`タグを使用し、少し凝ったデザインのアコーディオンを実装する。

### コードのポイント
- `<hgroup>`など、適切なタグを使用したセマンティックなHTML。
- 適切なariaの属性付与によるアクセシビリティ対応。
